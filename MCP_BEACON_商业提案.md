## 一、做什么

**一句话**：AI Agent 的 Datadog —— 实时监控、告警、成本追踪。

**核心功能**：
1. **Agent 运行时监控** — 追踪每个 Agent 的 API 调用次数、延迟、Token 消耗
2. **异常检测** — Agent 陷入循环？幻觉率飙升？自动告警
3. **成本仪表盘** — 按 Agent/项目/模型 拆分费用，预算超限自动暂停
4. **日志聚合** — 多 Agent 日志统一搜索，错误自动归类
5. **性能优化建议** — 分析 prompt 效率，推荐更便宜的模型替代方案

## 二、为什么做

### 市场时机

2026 年是「AI Agent 大爆发」年。Claude Code、Cline、Cursor、Hermes Agent 等工具让每个开发者都跑着自己的 Agent。但有一个被忽略的问题：

> **没人知道自己的 Agent 在干什么、花了多少钱、什么时候会出错。**

就像 2014 年没人监控微服务 → 诞生了 Datadog/New Relic。2026 年没人监控 AI Agent → 这是一个空白。

### 市场规模

| 维度 | 数据 |
|------|------|
| 全球 AI Agent 开发者 | 300万+（2026预测） |
| 企业级 Agent 部署 | 年增长 200% |
| Datadog 市值参考 | $440亿（监控赛道天花板） |
| 可触达市场 | $2.1B |

### 竞争格局

| 竞品 | 价格 | 差距 |
|------|------|------|
| **LangSmith** | $39/月 | 只服务 LangChain，不通用 |
| **Helicone** | $0-599/月 | 只做 API 日志，不做 Agent 行为分析 |
| **Weights & Biases** | $0-几百/月 | 做模型训练，不做 Agent 运行 |
| **自建方案** | 工程师时间 | 碎片化，无统一标准 |

**空白**：没有一款面向「Agent 运行时」的通用监控平台。Helicone 最近但缺乏 Agent 行为分析（循环检测、幻觉监控、Prompt 优化）。

## 五、怎么做

### 30天路线图

| 周 | 任务 | 里程碑 |
|:---:|------|------|
| W1 | Landing Page + 用户认证 + SDK 发布 (npm包) | 开发者可以 `npm install mcp-beacon` |
| W2 | 核心监控：API 调用记录 + Token 计数 + 延迟追踪 | 实时仪表盘可用 |
| W3 | 异常检测：循环检测 + 幻觉率 + 告警 (Email/Slack) | 第一个告警发出 |
| W4 | 成本仪表盘 + 用户行为埋点 + 反馈收集 + 发布上线 | MVP 上线，开始积累用户 |

### 技术栈

```
前端: Next.js 15 + Tailwind + shadcn/ui
后端: Next.js API Routes
数据库: Supabase (PostgreSQL)
缓存: Upstash Redis
SDK: TypeScript npm 包 (主) + Python pip 包 (次)
部署: Vercel + Cloudflare
```

### 开发策略

**SDK 优先** — 核心是 npm 包 `mcp-beacon`。开发者只需：

```javascript
import { Beacon } from 'mcp-beacon';
const beacon = new Beacon({ apiKey: 'xxx' });
beacon.track(agent.run({...}));
```

SDK 自动拦截 HTTP 请求 → 记录延迟、Token、状态码 → 发回仪表盘。
