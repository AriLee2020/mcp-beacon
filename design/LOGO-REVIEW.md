## 设计决策记录

### 修改: V2 字母 R → B
- **原因**: 品牌名 "BEACON"，首字母应为 B 而非 R
- **争议**: 需求文档写"灯塔/R"，可能是早期代号
- **建议**: 如 R 有特殊含义（如 Radar），可回退，但需在评审中明确

### 设计原则对照 (DESIGN.md)
- ✅ 所有 Logo 使用品牌暖橙 #F97316
- ✅ 六边形作为统一 DNA
- ✅ 深色表面 #1A1A2E / #16213E
- ✅ 无装饰性渐变（v5 的片段渐变属于结构性使用）
- ⚠️ V4 文字色为 #1A1A2E（仅适用于浅色背景），需暗色变体

## 建议采用策略

**不单选其一，分层使用：**

1. **favicon (32px)** → V1 极简图标（已在 favicon.svg 使用）
2. **App Icon / 通用图标** → V1 或 V2 正方形图标
3. **网站 Header / 品牌主标** → V4 完整组合标
4. **品牌叙事 / 营销** → V3 灯塔意象
5. **技术文档 / 暗色界面** → V5 抽象科技
