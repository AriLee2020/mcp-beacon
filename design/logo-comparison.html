5 版方案，统一 DNA：六边形 + 信标 + 放射线 · 品牌色 #F97316 暖橙

📋 Designer 审阅意见

5 版方案覆盖了从极简图标到完整品牌锁定的不同场景。v1/v3/v5 偏向纯图标标记，v2 是字母标，v4 是完整品牌组合。建议根据使用场景（favicon/App Icon → v1；品牌主标 → v4）搭配使用，而非单选其一。

💡 Designer 建议：组合使用策略

建议不单选其一，而是按场景分层使用：
favicon / 小尺寸 Icon → V1（纯图标，32px 清晰可辨）
App Icon / 社交媒体头像 → V1 或 V2（方形识别度高）
网站 Header / 品牌主标识 → V4（完整组合标，含文字）
品牌故事 / 文档封面 → V3（灯塔意象，叙事性强）
暗色界面 / 技术场景 → V5（抽象科技感）
